Descrição Do Relatorio:

Este trabalho tem como objetivo explorar os conceitos de compressão de dados sem perdas tanto para imagens como áudio.

Introdução

No presente relatório, descreveremos o estudo realizado no segundo laboratório de codificação de sinais multimídia,
cujo principal tema aborda os algoritimos de compressão de dados sem perdas que foram testados utilizando dados de áudio e
imagem. O trabalhado foi dividido em tarefas especificas onde analisamos os códigos e tiramos conclusões sobre determinadas
caracaterísticas relacionadas a compressão dos dados. Os algoritimos modificaram os arquivos preparando-os para uma possível
compressão de dados , e também mostraram diversos resultados a apartir dessas mudanças realizadas. Resultados estes como
a qualidade dos dados, taxas de erro durante a modicação dos mesmos e outros aspectos relevantes.
